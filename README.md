# bb_visual
 Visualize 64bit bitboard.
 
## Dependencies
* Python3.5 and above
* Tkinter

## Features
* Mouse click toggles individual bits
* Synchronizes binary, hexadecimal, and board representations
* Supports all four types of endianness
* Basic bitboard operations
